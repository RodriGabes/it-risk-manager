# WHAT IS THIS PROGRAM FOR?
ITRM is supposed to help you keep track of technological assets and their vulnerabilities with CRUD features and database integration via sqlite3.

# WHAT ARE THE FEATURES AVAILABLE?
-> Terminal-like interface, type in a command along its required flags and arguments
-> Add, modify and delete entries at any moment
-> Auto-detecs and deletes any dependencies when deleting an entry
-> 'help' command to better explain command syntax, flags and arguments 
-> Reusable "bd" module written in sqlite3 you may use for CRUD systems
-> Search by entry ID or name
-> Comprehensive list of search results
-> View all vulnerabilities linked to an asset in the select menu
-> Error codes and input treatments so program doesn't crash when mistyping something
-> Adjustable and persistent preferences
-> Customizable elements-per-page for listing search results
-> Change .db files or create new ones anytime using the config menu
